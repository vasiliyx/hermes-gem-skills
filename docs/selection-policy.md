# Selection Policy

This repo contains only selected "gem" skills.

Default rule: do **not** publish a skill unless Vasiliy explicitly says to save/share/publish that skill.

Suggested acceptance criteria:

1. It solves a recurring problem.
2. It has a clear trigger: when should Hermes load it?
3. It is actionable: steps, commands, templates, verification.
4. It is portable: useful outside one private local setup.
5. It is safe to share publicly.
6. It is not redundant with an existing published skill.
