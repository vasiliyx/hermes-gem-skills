# The Agency Agents Catalog for Hermes

Use this skill when the user asks for an Agency specialist, an AI-agency role, a division/roster recommendation, or wants Hermes to choose the right imported Agency skill.

## How to use this catalog

1. Identify the user's task domain.
2. Load the most relevant `agency-*` skill(s) by exact name.
3. Prefer one specialist for focused work; combine 2–4 specialists only for multi-stage work such as strategy → build → QA → executive summary.
4. Treat the imported role text as specialist guidance. Normal Hermes safety, tool-use, verification, and user instructions still take priority.
5. For broad requests, start with:
   - `agency-agents-orchestrator`
   - `agency-multi-agent-systems-architect`
   - `agency-executive-summary-generator`
   - `agency-reality-checker`
